# Deloitte Data Analytics Job Simulation

A data analytics project completed as part of Deloitte's Data Analytics job simulation program (via Forage). The task involved working with telemetry data, performing data analysis in Excel, and building an interactive dashboard in Tableau to surface insights.

## 📖 About the Simulation

This simulation walks through a real-world style data analytics workflow:
- Reviewing raw telemetry data from connected devices/sensors
- Cleaning and structuring the data for analysis
- Applying analytical techniques to identify patterns and anomalies
- Visualizing findings through an interactive dashboard

## 📁 Files in This Repository

| File | Description |
|---|---|
| `daikibo-telemetry-data.json` | Raw telemetry dataset used for the analysis |
| `Task 5 Equality Table.xlsx` | Excel workbook with the analysis/equality table produced from the dataset |
